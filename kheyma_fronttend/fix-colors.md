# Color Theme Updates

The following color mappings should be used throughout the application:

## Light Mode Colors
- `text-gray-500` → `text-text-muted` (for muted text)
- `text-gray-400` → `text-text-muted` (for lighter muted text)
- `text-gray-600` → `text-text-muted` (for medium muted text)
- `bg-gray-100` → `bg-surface-light` or keep for subtle backgrounds
- `border-gray-200` → `border-border-light` or `border-gray-200` (acceptable)
- `border-gray-300` → `border-gray-300` (acceptable)

## Dark Mode Colors
- `text-gray-400` → `text-text-muted-dark` (for dark mode muted text)
- `text-gray-300` → `text-text-muted-dark` (for dark mode lighter text)
- `bg-gray-800` → `bg-surface-dark`
- `border-gray-700` → `border-border-dark` or `border-surface-border`

## Specific Color Replacements
- `#526355` → `text-text-muted`
- `#618968` → `text-text-secondary`
- `#9db9a1` → `text-text-muted-dark`
- `#111812` → `text-text-main` (already correct)
- `#f0f4f1` → `border-border-light`
- `#2a3c2e` → `border-border-dark` or `border-surface-border`
